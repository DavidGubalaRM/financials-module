# Financials Module - Technical Documentation

This folder contains the Docusaurus-based technical documentation for the Financials module.

## Prerequisites

- Node.js 18 or higher
- npm, yarn, or pnpm

## Setup

```bash
cd docs
npm install
```

## Development

```bash
npm run start
```

Opens http://localhost:3000 with hot reload.

## Build

```bash
npm run build
```

Output is in `build/` directory.

## Serve Built Site

```bash
npm run serve
```

Serves the built site locally (e.g., for verification before deployment).
